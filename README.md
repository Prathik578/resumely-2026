# Resumely

Resumely is an AI-powered resume evaluation platform that analyzes resumes from the perspective of a recruiter or HR professional and provides actionable feedback for improvement.

The project was developed using Lovable and modern web technologies as part of my exploration of AI-assisted software development and product building.

---

## Features

* Resume upload and analysis
* AI-powered evaluation
* HR-style feedback and recommendations
* Resume scoring and assessment
* Identification of strengths and weaknesses
* Suggestions for improvement

---

## Technologies

* Lovable
* React
* TypeScript
* Vite
* Tailwind CSS
* AI APIs

---

## Learning Objectives

This project was created to explore:

* AI-assisted software development
* Product design and development
* Modern web technologies
* Frontend and backend integration
* User experience design

---

## Project Goal

To provide students and job seekers with an accessible tool that delivers professional-style resume feedback and actionable suggestions for improvement.
